최적화 - Object Pooling 오브젝트 풀링

Pool : 보관함.

Object Pooling - 오브젝트를 재활용하는 최적화 기법.
1. 보관함(Pool) 만들기 // Queue로 구현해보기.
2. 미리 오브젝트를 '어느정도' 보관함에 만들어 놓고,
2-1. 오브젝트 생성 (Instantiate)
2-2. 오브젝트를 보관
2-3. 오브젝트를 초기화/비활성화


3. 오브젝터 생성 대신에 보관함에서 꺼내서 사용.
4. 오브젝트 파괴 대신 보관함에 보관.
총알에게 오브젝트 보관함의 위치를 알려줬다.(Initialize)

5. 총알이 만약 부족하면, 오브젝트를 새로 생성해야된다. CreateBullets()
오브젝트를 꺼낼 때 : 남아있는 오브젝트가 있는지 확인.(Count <= 0)
